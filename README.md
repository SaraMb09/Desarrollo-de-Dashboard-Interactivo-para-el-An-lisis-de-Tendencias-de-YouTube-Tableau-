# Desarrollo-de-Dashboard-Interactivo-para-el-An-lisis-de-Tendencias-de-YouTube-Tableau-
Desarrollé un dashboard interactivo para Sterling &amp; Draper, una agencia de publicidad, con el objetivo de automatizar el análisis de las tendencias de videos de YouTube y facilitar la toma de decisiones de los gerentes de planificación de campañas publicitarias.
